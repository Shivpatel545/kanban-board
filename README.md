# Team Task Management Board (Kanban Style)

## Description
A Kanban-style task management board built using HTML, CSS, and Vanilla JavaScript.

## Features
- Drag & Drop tasks
- Priority-based styling
- Task persistence using localStorage
- Search functionality

## Technologies Used
- HTML5 (Semantic Elements)
- CSS3 (Flexbox & Grid)
- JavaScript (Event Delegation, Arrow Functions)

## How to Run
1. Clone the repository
2. Open `index.html` in a browser
3. Start managing tasks!
